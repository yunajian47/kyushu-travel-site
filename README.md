# Kyushu Travel Site

This is a static website. Deploy the whole `kyushu-travel-site` folder to any static hosting service.

Recommended options:
- Netlify: drag this folder into Netlify Drop.
- Vercel: import as a static project.
- GitHub Pages: upload the folder contents and publish from the repository root.
- Cloudflare Pages: upload as a static site.

Entry file: `index.html`
Data file: `assets/places.json`
